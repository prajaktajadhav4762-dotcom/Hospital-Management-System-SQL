 Hospital Management System (SQL Project)

 Project Overview
This project is a SQL-based Hospital Management System designed to manage hospital data including departments, doctors, patients, appointments, and billing.

It demonstrates database design, relational modeling, and advanced SQL queries for analytical insights.

---

Database Schema Includes

- Departments
- Doctors
- Patients
- Appointments
- Billing

The database is designed using:
- Primary Keys
- Foreign Keys
- Proper relationships
- Indexing for optimization

---

Features & SQL Concepts Used

✔ Basic SELECT queries  
✔ Aggregate functions (SUM, COUNT, AVG)  
✔ GROUP BY & HAVING  
✔ INNER JOIN & LEFT JOIN  
✔ Subqueries  
✔ Views  
✔ Index creation  
✔ Window Functions (RANK, DENSE_RANK)  
✔ CTE (Common Table Expressions)  
✔ Date-based analysis  
✔ CASE statements  

---

Analytical Insights Implemented

- Total hospital revenue
- Department-wise revenue
- Doctor-wise revenue
- Highest earning doctor
- Top 3 earning doctors
- Doctors with no appointments
- Patients with multiple visits
- Monthly and daily revenue analysis

---
 How to Run

1. Run `schema.sql` to create tables
2. Run `insertdata.sql` to insert sample data
3. Run `queries.sql` to execute analytical queries
